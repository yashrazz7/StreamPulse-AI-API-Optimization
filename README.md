# StreamPulse: AI Model Performance & Cost Optimization Tower

### 📊 Project Overview
This project targets the critical challenge of rising enterprise LLM API costs and execution latency in 2026. Built using Tableau Desktop, it acts as a centralized control tower to monitor token consumption, Response Latency, and Cost Attribution across major AI providers (OpenAI, Anthropic, Gemini).

### 🛠️ Key Technical Features Inside
1. **Dynamic Tier Cost Attribution:** Custom calculated fields simulating multi-tenant enterprise API expenditures.
2. **Granular Performance Metrics:** Tracking Token Efficiency Ratio (TER) and Cost-to-Value Index (CVI).
3. **Responsive Grid Layout:** Cross-filtering enabled dashboard mapping department spend against system latency.
